MCQ File Format
===============


The MCQ question are saved in a csv file in the same directory as the
database text. In the :numref:`mcqFileExample`, you can see an example.

.. container:: center

   .. table:: MCQ File example
      :name: mcqFileExample

      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | A_Happy_Visitor.Beginner.txt             |                     |                                     |                           |                                  |   |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | Question Title                           | Answers             |                                     | Right Answer              |                                  |   |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | What is at the door of the house?        | a man               | a cat                               | a dog                     | a snake                          | 2 |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | What happens at the end of the story?    | Anna keeps the dog. | Anna takes the dog to the hospital. | Anna's mom feeds the dog. | Anna dries the dog with a towel. | 4 |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | Is the dog dry or wet?                   | The dog is dry.     | The dog is wet.                     | 2                         |                                  |   |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | How does the wet dog look?               | I only              | I and II                            | II and III                | I, II, and III                   | 1 |
      |                                          |                     |                                     |                           |                                  |   |
      | I. strong                                |                     |                                     |                           |                                  |   |
      | II. cute                                 |                     |                                     |                           |                                  |   |
      | III. small                               |                     |                                     |                           |                                  |   |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+
      | What day of the week is it in the story? | Monday              | Tuesday                             | Wednesday                 | Thursday                         | 4 |
      +------------------------------------------+---------------------+-------------------------------------+---------------------------+----------------------------------+---+


.. _mcqAnswer

MCQ Answer File
===============

The MCQ answers are saved in a csv file in the simulation directory. In the :numref:`mcqAnswerFileExample`, you can see an example.
