Action units

Action Units Management

For the Action Units, only 1, 2, 4, 5, 6, 7, 9, 10, 12, 14, 15, 17, 20, 23, 25, 26, 28, 45 Action Units are supported. Please see Action Units Wikipedia for more informations.

You can add your own group of Action Units, or the default group by clicking on “Add Default Action Units” button. See Action Units Default Wikipedia .

The action units are save in Eye Got It folder in User folder.
